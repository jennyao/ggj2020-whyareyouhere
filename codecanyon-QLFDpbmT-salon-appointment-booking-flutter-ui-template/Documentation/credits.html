  • custom_radio_grouped_button: Custom Radio Buttons and Grouped Check Box Button.
  • equatable: An abstract class that helps to implement equality without needing to explicitly override == and hashCode.
  • flutter_bloc: Widgets that make it easy to integrate blocs and cubits into Flutter.
  • flutter_is_emulator: Plugin to detect whether the current device is an emulator or a simulator.
  • flutter_markdown: A Markdown renderer for Flutter.
  • flutter_slidable: Slidable list item with directional slide actions that can be dismissed.
  • flutter_swiper: The best swiper(carousel) for flutter, with multiple layouts, infinite loop. Compatible with Android & iOS.
  • get_it: A simple Service Locator.
  • google_maps_flutter: A Flutter plugin for integrating Google Maps in iOS and Android applications.
  • image_picker: Flutter plugin for selecting images from the Android and iOS image library, and taking new pictures with the camera.
  • introduction_screen: Introduction/Onboarding package for flutter app with some customizations possibilities.
  • location: A Flutter plugin to easily handle realtime location in iOS and Android.
  • logging: Provides APIs for debugging and error logging.
  • logging_appenders: Logging appenders for the dart logging package for print, file and remote (logz, loki).
  • package_info: Flutter plugin for querying information about the application package, such as CFBundleVersion on iOS or versionCode on Android.
  • photo_view: A simple zoomable image/content widget.
  • pull_to_refresh: A widget provided to the flutter scroll component drop-down refresh and pull up load.
  • qr_flutter: Flutter library for simple and fast QR code rendering via a Widget or custom painter.
  • scrollable_positioned_list: A list with helper methods to programmatically scroll to an item.
  • share: Flutter plugin for sharing content via the platform share UI, using the ACTION_SEND intent on Android and UIActivityViewController on iOS.
  • shared_preferences: Flutter plugin for reading and writing simple key-value pairs. Wraps NSUserDefaults on iOS and SharedPreferences on Android.
  • shimmer: A package provides an easy way to add shimmer effect in Flutter project.
  • sprintf: Dart implementation of sprintf.
  • url_launcher: Flutter plugin for launching a URL on Android and iOS. Supports web, phone, SMS, and email schemes.
